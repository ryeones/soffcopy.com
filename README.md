# soffcopy.com

personal site for [soffcopy](https://www.soffcopy.com) - a media company for people still figuring it out.

written in obsidian. built with jekyll. hosted on github pages.

## structure

- `index.md` — home page
- `pages/about.md` — about
- `pages/studio.md` — soffcopy studio
- `pages/space.md` — soffcopy space
- `pages/fomties.md` — fomties
- `_layouts/default.html` — main layout
- `assets/css/main.css` — styles

## running locally
```bash
bundle exec jekyll serve
```

then open http://127.0.0.1:4000
